# Synthetic scenario — **Riverdale drill** (grounding pack for homework)

Use only these facts when writing or validating drill briefings. Facts are labeled **F1…** so validators can check traceability.

| ID    | Fact |
|-------|------|
| **F1** | The event is **Exercise Riverdale**, a **full-scale** county emergency exercise (not a real incident). |
| **F2** | **Location:** River County, unified staging at **Depot 7** (grid **RC-17**). |
| **F3** | **Window:** Exercise period **06:00–14:00 local** on the scheduled exercise day. |
| **F4** | **Activation:** EOC is at **Level 2 partial**; **three** departments committed liaison officers (Fire, EMS, Public Works). |
| **F5** | **Inject:** At **09:30**, sim-cell reports **simulated** chlorine odor near the **old mill** (exercise only; no real hazard). |
| **F6** | **Casualties:** **Zero** real injuries; all casualties are **role-players** per drill plan. |
| **F7** | **Comms:** **Primary:** 800 MHz county talkgroup **TAC-2**; **backup:** satellite phones issued to branch directors. |
| **F8** | **Public messaging:** Single county PIO statement scheduled for **11:00** (“exercise in progress, no danger to public”). |
| **F9** | **After action:** Formal AAR due **N+10 business days** to the county EM office. |
| **F10**| **Logistics constraint:** Potable water bladders **not** deployed in this scenario version (budget tabletop only). |

**Prohibited claims** (must not appear in a high-quality briefing unless explicitly labeled *training fiction*): real hazmat release, mass casualties, bridge collapse, federal disaster declaration.
